# 1. 功能说明
   1. 软件模拟串口实现与ssp737x通信。
   2. 代码基于SS881A编写测试，时钟12M等待周期4

# 2. 功能实现
- ssp737x的基本驱动程序
- 复位ssp737x并进入MCU模式
- 单击按键开关ssp737 boost
